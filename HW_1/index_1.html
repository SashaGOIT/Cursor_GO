Basic comands

  • git config -- global user.name Sasha задатиє ім'я користувача в системі (тільки один раз);
  • git config -- global user.email andriyovichs9@gmail.com задатиє email користувача в системі (тільки один раз);
  • git init -- створення нового сховище для зберігання;
  • git status -- відображення статусу оновлення;
  • git add -- додає зміни із робочого простору в проіндексованих файлів;
  • git commit -- команда для записування проведених змін;
  • git log -- команда яка показує список усіх комітів;
  • git checkout -- використовується для переключення гілок та отримання вмісту до вашої робочої директорії;
  • git branch -- список усіх гілок, створення нових, видаляти і перейменовувати їх;
  • git push -- завантажує нову гілку на репозиторій;
  • git pull -- завнатажує коміти, файли і силки з видаленого репозиторію у локальний репозиторій;
  • git merge -- злиття однієї або декількох гілок в одну;
  • git clone -- створення нових каталогів.

Gitignore - текстові файли які можна відкрити за допомогою будь-якої програми.

Comands Advanced

  • git revert - дозволяє скасувати будь-які коміти у репозиторію;
  • git reset - зміна локальних змін в репозиторії;
  • git rebase - дозволяє взяти всі зміни, що були в комітах одної гілки та відтворити їх на іншій гілці;
  • git cherry-pick - команда, яка дозволяє вибирати довільні коміти Git за посиланням і додавати їх;
  • git stash - чернетка для змін.
